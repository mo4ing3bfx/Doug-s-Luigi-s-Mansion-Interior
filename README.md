An interior mod from a guy who just really enjoys Luigi's Mansion :)

Development of this mod is currently ongoing, and in its current state this mod only adds an interior.
This is my first time making a mod for Lethal Company, so please do not expect perfection. I've done my
best to iron out the kinks as best I can, but I'm certain there will still be issues and will address them as 
they are discovered.

I highly recommend using this mod with PanMan's LuigisMansionMoon LLL mod: https://thunderstore.io/c/lethal-company/p/PanMan/LuigisMansionMoon_LLL/
as it adds plenty of custom LM themed scrap items! Plus, it just looks really cool.

You can configure the interior's weights to spawn on any moon in the config file, but by default it follows the same spawn weights as the "Manor" interior.
It's also set up to always spawn for PanMan's moon, but if you'd prefer other interiors for that moon to spawn with it you can change that in the config as well.

*Note: If you use PanMan's moon, I've noticed issues with being able to load in on the 1.7.8 update of Lethal Level Loader. If
you want to use that moon, it still works on 1.7.7*
